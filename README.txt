How to setup the project?

1) Open project in IDE(eclipse)
2) build maven

2) If maven fails:
- Add muse classes to library.(It's in lib folder)
- Add guava classes to library.
- Add apache class to library
Files need to be added to both java and web library.

3) change the file path in incumbency_table with the file path of csv
4) run incumbency_table on server.
5) open http://localhost:8080/incumbency/index.html
